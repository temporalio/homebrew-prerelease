# Temporalio Prerelease

## How do I install these formulae?

`brew install temporalio/prerelease/<formula>`

Or `brew tap temporalio/prerelease` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "temporalio/prerelease"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
